# vue-router-nav

> A vue.js router navigation bar plugin based on vue-router.

## Install
`npm install vue-router-nav --save`

## Usage

``` javascript
import RouterNav from 'vue-router-nav'

Vue.use(RouterNav)
```

``` html
<template>
  <div id="app">
    <router-nav/>
    <router-view/>
  </div>
</template>

<script>
export default {
  name: 'app'
}
</script>
```
